Principais formatações

Negrito, usando a tag strong, que não funciona em títulos (tag h).

Itálico, usando a tag "em" (emphasis).

Podemos adicionar uma tag por cima de um texto usando ctrl+shift+p, clicando em "Wrap with abreviation", e logo em seguida inserindo a tag que irá abraçar o texto.

Texto marcado

Podemos marcar texto usando a tag Mark

Texto sobrescrito

Para inserir coisas do tipo x20+3; usa-se a tag sup.

Testo subescrito

Para inserir coisas do tipo H2O; usa-se a tag sub

Texto riscado

Para inserir o texto riscado, usa-se a tag del

Texto sublinhado

Para inserir o texto sublinhado, usa-se a tag ins

Abreviações

Para exibir um significado para algo que você escreveu, use a tag abbr

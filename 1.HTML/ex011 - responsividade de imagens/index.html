Imagens dinâmicas

Imagens dinâmicas são imagens que se adaptam ao formato da tela do usuário, elas são importantíssimas para uma boa experiência ao acessar seu site.

Aqui vai um exemplo de imagem dinâmica, ela se adapta em diferentes dispositivos, ou se o usuário muda o tamanho da janela.

Logo após a imagem, leia sobre a tag picture

Para fazer essa adaptação, foi preciso criar 3 imagens com tamanhos diferentes.
(Todas com 80 px/pol ou px/in de resolução)

  1. Uma pequena com 300 px
  2. Uma média com 700 px
  3. Uma grande com 1000px

Tag picture

Para criar uma imagem que se adapta automaticamente a tela, é necessário usar a tag picture. Dentro dela é necessário colocar a tag img,ou seja,
envelopar a tag img dentro da tag picture ficando assim: 

<picture>
    <img src="imagem" alt="texto-alternativo">
</picture>

Logo após isso, deve-se inserir dentro do envolpe picture, a tag source:media:type, que irá conter a largura máxima, o tipo da imagem, e o endereço da imagem que vai se adaptar. Ficando assim: 

<picture>
    <source media="(max-width: TAMANHO EM MÁXIMO EM PIXELS PARA ACONTECER A ADAPTAÇÃO DA IMAGEM)" srcset="ENDEREÇO DA IMAGEM" type="image/TIPO DA IMAGEM">        
    <img src="imagem" alt="texto-alternativo">
</picture>

Detalhes:

  • A imagem padrão na tag img tem que vir ABAIXO no código, e acima dele devem ser inseridos as imagens alternativas na tag source, do menor para o maior.
  • O tamanho máximo em pixels deve conter um espaço no início, e finalizar em "px", sem espaço entre os números e o "px": media="(max-width: 750px)"
  • O nome do arquivo de imagem não pode conter espaços, acentuação, ou letras maíusculas e deve conter o formato do arquivo no final com ".": srcset="imagens/imagem-pequena.png"
  • O tipo do arquivo nao deve conter ".": type="image/png"

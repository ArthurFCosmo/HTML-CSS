Inserindos vídeos hospedados localmente

Maneira simples

Esse vídeo está hospedado no meu próprio servidor (pasta).

Foi utilizada a tag video, que funciona de maneira muito semelhante a tag audio.

<video src="CAMINHO DO VÍDEO" autoplay muted controls></video>

Parâmetros

  • autoplay - Reproduz o vídeo automaticamente ao abrir a página (Está indisponível na maioria dos navegadores, mas FUNCIONA caso o parâmetro MUTED esteja ativado).
  • muted - Retira o som do vídeo quando ele é iniciado.
  • controls - Mostra os controles (play, volume, tela cheia, etc.).
  • poster - Permite upload de uma thumb (capa).
  • height - Configura o espaço que o vídeo ocupa na página (altura).
  • width - Configura o espaço que o vídeo ocupa na pagina (largura)

Maneira correta

Para se inserir da maneira correta, deve-se utilizar a tag video, apagar o src="#" original, e inserir dentro da tag video a tag source:src, que pode ser adicionada várias vezes para adicionar vários formatos do mesmo vídeo, maximizando a compatibilidade. Ficando assim: 

<video height="ALTURA PERSONALIZAVEL" controls poster="THUMB PERSONALIZAVEL">
    <source src="media/video-naum.mp4 (local do arquivo)" type="descubra o mediatype no site iana">
    <source src="media/video-naum.ogv (local do arquivo)" type="descubra o mediatype no site iana">
    <p>Seu navegador não tem compatibilidade com reprodução vídeos (mensagem caso nenhum arquivo consiga ser lido)</p>
</video>

Código-fonte

O comando document.getElementById('teste) é escrito em linguagem JavaScript, com o o parametro code (código-fonte), você pode colocar o texto numa fonte em que todas as letras sao do mesmo tamanho.

Já com o parametro 'pre' (pre-formatação), você consegue preservar a formatação original do texto, como demonstrado a seguir:

        
            num = int(input('Digite um número:'))
            if num % 2 == 0:
                print('O número {num} é ímpar.')
            else:
                print('O número {num} é par.')
            print('Fim do programa.')

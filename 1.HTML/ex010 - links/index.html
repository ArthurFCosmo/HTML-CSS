Usando links externos

Um link externo é link que vai apontar para outro site (que não é meu)

Para criar um link usa-se a tag a (anchor/âncora), e nela existe o parametro obrigatório href="#", onde deve ser inserido o link.

Abrindo em uma nova aba

Pode-se observar que ao clicar no link anterior, a página original foi sobrepujada. Para abrir o link em uma nova aba automaticamente, usa-se o parametro da tag a:
target="_blank", e o parametro rel="external", quando rel (relativo a) recebe o valor "external", indica ao mecanismo que aquele link se refere a uma página externa.

Você pode acessar meu canal do youtube

Para abrir o conteúdo na própria página, basta não declarar target="#", ou declarar target="_self".

Nofollow

Quando você quer inserir um link no seu site, mas não se responsabiliza pela gerência daquele site, ou não da uma aval para aquele conteúdo. Você deve usar o parâmetro rel="nofollow", e existe o parâmetro rel="dofollow", que serve para indicar justamente o contrário.

Link com nofollow: Clique aqui

Isso serve para informar ao mecanismo de busca, e não para o usuário.

Links para download

Para criar um link para download, você deve usar a tag a, com o parâmetro href="#" indicando o arquivo normalmente, porém deve adicionar o parâmetro download="#", com o nome do arquivo, e o parâmetro type="#", esse parâmetro tem um valor que depende do tipo de arquivo que você está tentando disponibilizar para download. Para descobrir esse valor acesse www.iana.org.

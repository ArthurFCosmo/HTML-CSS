Variáveis

As variáveis em CSS funcionam como nas linguagens de programação. Elas armazenam dados, para que possam ser acessados de maneira mais fácil posteriormente.

Para declarar uma variável se usa :root {}

Root e var:

Deve-se declarar a variável no seletor root, da seguinte forma:
--nome da variavel: valor da variavel
E para chama-la depois se usa var(--nome da variavel)

Seletor global

Para selecionar todas as CSS, pode se utilizar o seletor global * {}

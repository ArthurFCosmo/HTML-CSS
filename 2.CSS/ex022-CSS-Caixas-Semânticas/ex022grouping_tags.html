Grouping Tags

O que são

Grouping tags são tags html, elas funcionam como caixas comuns. No entanto, elas servem para organizar o site semanticamente, fazendo com que o navegador reconheça a natureza de cada elemento.

HTML4

Antes se usamvam diversas divs, e para torna-las únicas, se usava o estilo "id". Colocar "id" em uma grouping tag pode parecer redundante, mas não é proibido. Um uso interessante é nas tags "section".

As tags em si

  • Header: Cabeçalho
  • Main: Conteúdo Principal
  • Footer: Conteúdo de rodapé
  • Section: Secção
  • Article: Artigo, geralmente de texto
  • Aside: Conteúdo periférico (ex:nome do autor de um texto)
  • Nav: Menus de navegação

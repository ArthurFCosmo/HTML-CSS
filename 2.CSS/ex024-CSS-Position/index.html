Position

O que é

Position é um estilo CSS que configura a posição de um elemento em relação a outro.

Position: static;

O estilo padrão é o static.

Os elementos posicionados com essa configuração não são afetados pelas propriedades top, bottom, left, right.

É posicionado comumente, sem nenhuma propriedade especial. Sempre é posicionado seguindo o fluxo da página.

Position: relative;

O estilo relative pode receber várias configurações, como top, bottom, left, right.Todas essas configurações tem como base a posição normal que o item teria na página.

Se o elemento for mudado de lugar, nenhum outro elemento se moverá para preencher o espaço deixado por ele. É como se ele ainda estivesse ali.

Position: fixed;

Quando o estilo fixed está ativo, o objeto se alinhará a viewport, mantendo sua posição independente do scroll. Usa-se top, bottom, right, left.

Os elementos fixed não deixarão espaço na página onde eles estariam posicionado normalmente.

Position: sticky;

Um elemento com position: sticky é posicionado com base no scroll do usuario. Ele altera suas propriedades entre position: relative; e position: fixed; de acordo com o scroll.

Ele tem propriedades de position: relative; até que uma posição predefinida é encontrada na viewport, então o elemento "gruda" no lugar, como se tivesse a propriedade position: fixed;

Você tem que especificar pelo menos uma posição top, bottom, left, right; para a função sticky funcionar.

Position: absolute;

Quando um elemento está com position absolute, ele é posicionado com base no elemento pai mais próximo, ao invés de tomar como base a viewport. Se ele não tiver elemento pai, ele se baseará no body.

Se move de acordo com o scroll, ficando em um lugar fixo na página.

É removido do fluxo normal da página, e pode sobrepor outros elementos.

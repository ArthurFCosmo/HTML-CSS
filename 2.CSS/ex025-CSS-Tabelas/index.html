Tabelas em HTML5

Tags

Para criar tabelas em HTML usa-se a tag table, dentro dela você irá inserir linhas (table row) com a tag tr, e nessas linhas, você irá inserir dados (table data) com a tag td. O fechamento dessas tags é opcional, mas o vscode já fecha automatciamente as tags.

Ex:

Resultando nesta tabela, sem estilo algum:

A1 B1 C1
A2 B2 C2
A3 B3 C3

Podemos adicionar estilo nas tabelas.

Resultando em:

A1 B1 C1
A2 B2 C2
A3 B3 C3

Alinhamento vertical e horizontal

  • text-align: left, center, right
  • vertical-align: top, middle, bottom
